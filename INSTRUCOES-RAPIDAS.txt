RN CRM Vendas — versão NPM/React Clean

1. Suba no GitHub os arquivos desta pasta.
2. Não suba: node_modules, dist, .env e package-lock.json.
3. No Netlify use:
   Base directory: deixe em branco
   Build command: npm run build
   Publish directory: dist
4. Cadastre no Netlify:
   VITE_SUPABASE_URL
   VITE_SUPABASE_ANON_KEY
   SUPABASE_SERVICE_ROLE_KEY
5. No Supabase, rode supabase/schema.sql.
6. Crie o admin no Supabase Auth.
7. Rode supabase/admin_setup.sql.
8. Faça deploy no Netlify.

Visual atualizado para padrão clean branco e profissional.

CORRECAO IMPORTANTE:
Se aparecer erro falando que a coluna phone nao existe em profiles, rode no Supabase o arquivo:
supabase/ATUALIZAR-BANCO.sql
Depois refaça o teste de cadastro de vendedor.
