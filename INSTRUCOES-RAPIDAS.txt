RN CRM Vendas - WhatsApp Cloud API

1) Suba esta versão no GitHub.
2) Netlify:
   Build command: npm run build
   Publish directory: dist
   Base directory: deixe em branco

3) Cadastre no Netlify as variáveis:
   VITE_SUPABASE_URL
   VITE_SUPABASE_ANON_KEY
   SUPABASE_SERVICE_ROLE_KEY
   WHATSAPP_ACCESS_TOKEN
   WHATSAPP_PHONE_NUMBER_ID
   WHATSAPP_VERIFY_TOKEN
   WHATSAPP_GRAPH_VERSION=v20.0

4) Supabase:
   Rode supabase/schema.sql se for banco novo.
   Se já existe, rode supabase/ATUALIZAR-BANCO.sql.
   Crie o admin e rode supabase/admin_setup.sql.

5) Na Meta/WhatsApp Cloud API:
   Callback URL:
   https://SEU-SITE.netlify.app/.netlify/functions/whatsapp-webhook

   Verify Token:
   O mesmo valor cadastrado em WHATSAPP_VERIFY_TOKEN no Netlify.

6) Assine o evento messages.

7) Quando um cliente mandar mensagem para o WhatsApp da RN Vision Pira, a conversa aparece em Atendimentos WhatsApp.
