RN CRM Vendas - Negociação via WhatsApp do Vendedor

1) Suba no GitHub os arquivos desta pasta.

2) Não suba:
   node_modules
   dist
   .env
   package-lock.json

3) No Netlify:
   Base directory: deixe em branco
   Build command: npm run build
   Publish directory: dist

4) Variáveis no Netlify:
   VITE_SUPABASE_URL
   VITE_SUPABASE_ANON_KEY
   SUPABASE_SERVICE_ROLE_KEY

5) No Supabase:
   Se for banco novo, rode supabase/schema.sql.
   Se já usa o CRM, rode supabase/ATUALIZAR-BANCO.sql.

6) Como funciona a negociação:
   O vendedor seleciona o lead, clica em Iniciar WhatsApp e conversa pelo WhatsApp dele.
   O CRM salva apenas o histórico que ele registrar: resumo, objeção, proposta, retorno e status.

7) Depois do deploy, remova o app/atalho antigo e instale novamente se quiser atualizar o ícone/PWA.
