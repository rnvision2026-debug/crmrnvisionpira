RN CRM Vendas - Negociações simplificadas

1. Extraia o ZIP.
2. Suba os arquivos no GitHub.
3. Não suba node_modules, dist, .env ou package-lock.json.
4. No Netlify configure:
   Build command: npm run build
   Publish directory: dist
   Base directory: deixe em branco
5. Configure as variáveis do Supabase no Netlify.
6. Rode no Supabase o arquivo supabase/ATUALIZAR-BANCO.sql.
7. Faça Deploy project without cache.

Novidade:
- Admin altera a mensagem automática em Negociações.
- Use variáveis: {vendedor}, {cliente}, {empresa}, {servico}, {valor}.
